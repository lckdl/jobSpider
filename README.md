# jobSpider
Learning Python &amp; Scrapy

Mongodb required.
### Installation
```
$ git clone https://github.com/lckdl/jobSpider.git
$ cd jobSpider
$ pip install -r requirements.txt
```

### Run spider
```
$ python run.py
```
or
```
$ scrapy crawl lagou -a key=[keyword]
```

### Run visualization
```
$ python display.py
```
